# 2026 第五届"问途杯"决赛规则及评分指南

## 一、决赛内容

### (一) 决赛项目

决赛项目为**"酒店/旅游业智能客服应用功能整合优化与方案路演"**。

参赛队通过竞赛平台搭建具备自主预订功能的智能 Agent（智能体），并将其与初赛阶段完成的智能客服应用进行整合与优化设计，然后将智能客服应用设计与整合优化的整体思路形成路演 PPT 方案提交。最后根据已提交的路演 PPT 方案进行线下路演。

### (二) 关键时间节点

| 事项 | 时间 |
| :--- | :--- |
| 竞赛平台开放 | 5月7日 17:00 |
| 智能客服预订功能整合作品提交截止 | 5月25日 17:00 |
| 路演 PPT 提交截止 | 5月25日 17:00 |
| 线下路演及颁奖典礼 | 5月30日–5月31日 |
| 教学研讨会及校企合作论坛 | 5月31日 |

### (三) 决赛考核分值比重

| 考核模块 | 考核方式 | 权重 |
| :--- | :--- | :--- |
| 酒店/旅游业**智能预订 Agent 搭建** | AI评分 + 专家评审 | **10%** |
| 酒店/旅游业**智能客服应用功能整合及优化设计** | AI评分 + 专家评审 | **30%** |
| 酒店/旅游业**智能客服应用设计与整合优化方案路演** | 专家评审 | **60%** |

### (四) 决赛具体考核要求

#### 1. 智能预订 Agent 搭建
- 针对核心交易场景，搭建具备**自主预订功能**的智能 Agent
- 深入研究预订业务的复杂规则，设计精准的**工具调用与任务规划机制**
- Agent 需能自主处理：客人预订需求分析 → 房态/票务查询 → 预订流程引导 → 信息核验
- 实现从"智能问答"向"**智能服务**"的功能跨越

#### 2. 智能客服应用功能整合及优化设计
- 将智能预订 Agent 与初赛智能客服应用**功能整合与深度集成**
- 对应用的设计与交互逻辑进行整体调优
- 使功能从单纯信息交互延伸至**复杂业务处理**

#### 3. 路演 PPT 要求
- 汇报时间：**10分钟**；专家答辩：**5分钟**
- 深度融入 **OPC 经营理念**（一人公司/极简商业单元）
- PPT 格式：`.pptx`（需同时提交 PDF 版本）

**路演 PPT 必须包含以下模块：**
1. 智能客服需求分析与整体规划设计
2. 智能客服知识库资料整理与构建逻辑
3. 智能客服流程设计与功能模块规划
4. 智能客服应用编排与关键节点配置
5. 智能客服应用性能展示与成本优化策略
6. 智能预订 Agent 搭建与智能客服功能整合
7. 智能客服应用设计优化与性能优化表现

## 二、决赛评分标准（详细）

### 模块一：智能预订 Agent 搭建（10分）

| 评分指标 | 细则 |
| :--- | :--- |
| 搭建与编排 | 酒店客房/景区门票在线销售信息设置正确，可满足预订需求 |
| | 预订 Agent 搭建完整，各要素正确，可实现实时查询房态/票务、信息提取与校验并完成预订 |
| | 模型选择及提示词设计合理，工具调用逻辑与预订流程引导清晰，客户体验流畅 |
| Agent 性能 | 能准确识别用户意图，工具调用逻辑清晰准确 |
| | 遇到工具调用失败、参数缺失、超出业务权限等异常时，具备**合理的逻辑纠偏与引导处理机制** |

### 模块二：智能客服应用功能整合及优化设计（30分）

| 评分指标 | 细则 |
| :--- | :--- |
| 功能整合 | 智能客服成功整合了预订功能，用户可在同一应用中实现**业务咨询与业务预订的双重需求** |
| 设计优化 | 整合后设计合理高效，能有效解决客服场景下**业务转化痛点**，具备实用性与落地性 |
| | 整合后兼具**轻量化与高效率**，提升复杂交互场景下的响应速度与稳定性 |
| 体验优化 | 能精准识别用户咨询过程中的**复杂意图变化**，准确记忆上下文关键信息 |
| | 在确保交互自然流畅的同时**控制 token 消耗成本**，展现真实运营场景下的经济优势 |

### 模块三：路演方案（60分）

| 评分维度 | 分值 | 评分要点 |
| :--- | :--- | :--- |
| PPT 设计与制作 | 10分 | 价值观正确；内容组织清晰反映完整策划思路；设计简洁美观、排版专业；视觉化处理清晰直观 |
| 方案汇报 | 40分 | **深度融入 OPC 经营理念**（核心！）；体现对业务场景与 AI 技术的理解；准确传达核心价值和设计亮点；具备实用性与落地性；体现创新型和独特性；汇报思路清晰、表达流畅 |
| 现场答辩 | 5分 | 观点明确有理有据；体现团队解决复杂问题的综合能力和高级思维 |
| 团队面貌 | 5分 | 着装得体、准备充分；分工合理、配合默契 |
