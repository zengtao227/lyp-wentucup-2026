## 一、核心景区

### 1.1 小鸟天堂

| 项目 | 内容 |
|:---|:---|
| 距酒店距离 | 5公里 |
| 级别 | 全国最大的天然赏鸟乐园之一 |
| 特色 | 以独特鸟类生态风景为主题，集生态旅游、文化旅游、健康旅游、休闲旅游于一体 |
| 历史 | 1984年，巴金先生亲笔题写"小鸟天堂"四字 |
| 主要景点 | 水乡风情展馆、鸟博馆、鸟趣园、生态馆（融声、光、电及开放式展出） |
| 位置 | 位于距广州市100千米外的江门市新会区天马村天马河的河心沙洲上 |

### 1.2 圭峰山国家森林公园

| 项目 | 内容 |
|:---|:---|
| 距酒店距离 | 6.4公里 |
| 级别 | 国家4A级旅游景区、国家级森林公园 |
| 面积 | 55.1平方公里 |
| 最高峰 | 云峰海拔545米 |
| 五大核心区域 | 圭峰山、开湖、周恩来纪念馆、莹博园和运动公园 |
| 特色 | 自然景观与人文景观融合，山体形似圭璧，森林覆盖率极高 |
| 主要景点 | 玉台寺（建于唐代，为广东四大丛林之一，位于南半山腰）、圭峰叠翠、枫林秋色 |

### 1.3 梁启超故居

| 项目 | 内容 |
|:---|:---|
| 距酒店距离 | 5公里 |
| 位置 | 广东省江门市新会区会城街道茶坑村中村二巷8号 |
| 建筑年代 | 晚清（1875-1908年） |
| 建筑面积 | 412平方米 |
| 构成 | 故居、怡堂书室 |
| 故居布局 | 一厨房、一门楼、两天井、一饭厅、一正厅、两耳房 |
| 藏书阁 | 门楼侧有梯级直达顶部的藏书阁 |
| 意义 | 梁启超童年接受启蒙教育和少年时期成长生活的地方 |

### 1.4 新会陈皮村

| 项目 | 内容 |
|:---|:---|
| 距酒店距离 | 6.4公里 |
| 定位 | 集新会陈皮产业服务、特色餐饮、文化旅游于一体的特色产业旅游综合体 |
| 核心 | 以新会陈皮为核心，实现多业融合 |
| 特色 | 新会陈皮国家现代农业产业园三产融合示范区及核心发展项目 |
| 设施 | 陈皮文化体验馆（国内首个国家地理标志产品多媒体体验馆）、陈皮交易市场 |

### 1.5 叱石风景区

| 项目 | 内容 |
|:---|:---|
| 历史 | 距今400多年历史 |
| 海拔 | 380米 |
| 缆车 | 2015年2月，叱石景区观光缆车建成投入运营 |
| 特色 | 天然森林氧吧 |
| 主要景点 | 叱石松涛、羊石坑、观音岩、黄大仙祠、叱石寺、"叱石成羊"、"天然石室"、"一洗红尘" |

### 1.6 梅江农业生态园

| 项目 | 内容 |
|:---|:---|
| 距酒店距离 | 1.6公里 |
| 占地面积 | 约428亩 |
| 碧道系统 | 1.8公里 |
| 建设 | 一期2021年7月开放，二期2023年1月18日建成 |

## 三、网红打卡地

### 三十三墟街

| 项目 | 内容 |
|:---|:---|
| 位置 | 江门市蓬江区 |
| 名称由来 | 连接茵江河岸与墟顶的33级青石板梯 |
| 历史 | 建于清朝光绪年间，元朝前滩涂演变形成商贸城集 |
| 历史建筑 | 墟顶街、余庆里等，完整保留岭南骑楼建筑特色 |
| 建筑风格 | 融合罗马柱、穹顶等多元风格 |
| 热度 | 2023年因电视剧《狂飙》热播成为网红打卡地，日均游客量显著提升 |

## 五、景区通用信息

### 5.1 购票方式

景区门票通过 **「小途乐游」** 小程序购买。

### 5.2 周边交通提示

- 多数景点距离酒店5-10公里范围内
- 建议出行方式：自驾、出租车或网约车
- 高铁出行：江门站（距酒店约3公里）
